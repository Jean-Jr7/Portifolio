Sobre mim

Desenvolvedor backend em Java com experiência prática em desenvolvimento web utilizando Java para criar APIs RESTful, aplicativos JSP e JSF. Conhecimento sólido em persistência de dados utilizando JPA e Hibernate, garantindo eficiência e integridade do banco de dados. Familiaridade com o framework Spring para desenvolvimento ágil e modular de aplicações Java. Competência em design e consulta de bancos de dados SQL para manipulação eficiente de grandes conjuntos de dados. Experiência em desenvolvimento orientado a testes (TDD), assegurando a qualidade e robustez do código. Habilidade em front-end com JavaScript, HTML e CSS para criar interfaces responsivas e agradáveis ao usuário. Conhecimento básico em serviços em nuvem AWS, proporcionando experiência adicional em ambientes escaláveis e distribuídos.

Projetos

Sistema de Gerenciamento de Biblioteca

O Sistema de Gerenciamento de Biblioteca é uma aplicação Java desenvolvida para facilitar o gerenciamento de uma biblioteca. Ele permite que os funcionários da biblioteca realizem várias operações, como adicionar novos livros ao acervo, cadastrar novos usuários, emprestar e devolver livros, gerar relatórios de empréstimos e muito mais

Tecnologias Utilizadas: Java, MySQL, Spring Framework (para desenvolvimento web e injeção de dependências), JavaFX (para criar uma interface de usuário desktop), Maven (para gerenciamento de dependências), Hibernate (para mapeamento objeto-relacional)
